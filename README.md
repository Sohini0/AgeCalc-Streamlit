#  Smart Age Calculator (Streamlit App)

This is a simple and interactive **Age Calculator Web App** built using **Python** and **Streamlit**.
The application allows users to enter their birth date and instantly see interesting details about their age.

---

## Features

*  Calculate **age in years, months, and days**
*  Show **total days lived**
*  Display **age in hours**
*  Show **days remaining until next birthday**
*  Special **birthday celebration animation**

---

## 🛠️ Technologies Used

* Python
* Streamlit
* Datetime module

## ▶️ How to Run the App Locally

1. Clone the repository

```
git clone https://github.com/yourusername/streamlit-age-calculator.git
```

2. Navigate to the project folder

```
cd streamlit-age-calculator
```

3. Install dependencies

```
pip install -r requirements.txt
```

4. Run the Streamlit app

```
streamlit run app.py
```

The app will open in your browser at:

```
http://localhost:8501
```

---

## 🌐 Deployment

This application can be deployed easily using **Streamlit Community Cloud**.

Steps:

1. Upload project to GitHub
2. Connect GitHub repository to Streamlit Cloud
3. Deploy `app.py`
